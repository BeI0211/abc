# FocusFlow 제품 기획서 (v1.0)

## 1. 개요
'FocusFlow'는 기록에 드는 에너지를 최소화하고 오직 실행에만 집중할 수 있는 환경을 제공하는 개인용 할일 관리 애플리케이션입니다.

## 2. 핵심 목표
- **Zero-Friction**: 할일 입력 완료까지 3초 이내 단축.
- **성공 지표**: 런칭 3개월 내 WAU 1만 명, 유료 전환율 3% 달성.

## 3. 타겟 사용자
- 바쁜 일상 속에서 즉각적인 기록이 필요한 30대 직장인.

## 4. 핵심 기능 (P0/P1)
- **Quick Add (P0)**: 어디서든 즉시 입력 가능한 퀵 애드 인터페이스.
- **스마트 태깅 (P0)**: 해시태그(#)를 활용한 자동 분류 시스템.
- **지능형 알림 (P1)**: 마감 직전 및 설정 시간 맞춤형 푸시 알림.

## 5. 디자인 방향성
- **Minimalism**: 불필요한 설정을 배제한 깔끔한 UI.
- **Efficiency**: 키보드 단축키 및 드래그 앤 드롭 기반의 빠른 조작.
- **Focus**: 현재 집중해야 할 일 위주의 화면 구성.
